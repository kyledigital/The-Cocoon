# The Cocoon

Simple static landing page for The Cocoon by Kyle Hector.

## Stack

- Plain HTML and CSS
- Hosted on Vercel
- Custom domain: `thecocoon.kylehector.com`

## Main Files

- `index.html` - production homepage served at `/`
- `.vercel/project.json` - local link to the Vercel project

## Vercel Settings

- Framework Preset: `Other`
- Root Directory: `.`
- Build Command: none
- Output Directory: none

## Deployment

This repository is connected to Vercel Git integration.
Pushes to `main` should trigger a production deployment automatically.
